注意，嵌套的if语句是以非常自然的方式自动处理的。首先，语法是递归表达的。aLineSequence可以包含IfStatments，IfStatments包含LineSequences。其次，这会导致语法处理方法以递归方式相互调用。因此，语法元素的嵌套被转换为递归方法调用。
